Pesquisa

Foi realizado uma pesquisa para levantar dados socioeconômico dos alunos do curso de Análise e Desenvolvimento de Sistemas da Fatec Franca para uma atividade da matéria de Sistemas de Informação, foi utilizando o Google Forms para levantamento dos dados sendo que o formulário foi respondido pela maioria dos alunos dos dois períodos (matutino e noturno). O conteúdo coletado na pesquisa foi processado pelo grupo responsável por este website utilizado para desenvolvimento ASP.net MVC, banco de dados SQL Server, e Microsoft Power BI para exposição dos resultados. No upload do arquivo ocorre uma leitura de todas as colunas para validação e tratamento dos dados. O trabalho está sendo apresentado na data de hoje (11 de setembro de 2017) para a turma do período noturno.
